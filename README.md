# Linux Terminal Commands

&lt;command&gt; &lt;option&gt; &lt;arguments&gt;

## Some Commands

**Changing password**

```shell
passwd
```

---

**For System Exit**

```shell
exit or
logout or
Control + D

```

---

**System Variables**

```shell
set
```

---

**List files and folders**

```shell
ls
```

_some options_

```shell
ls -a
ls -l
ls -al
ls -a /usr/folder
```
