# Linux Terminal Komutları

&lt;Komut&gt; &lt;seçenek&gt; &lt;argüman&gt;

**Komutların kullanma kılavuzu için**

man &lt;komut&gt;

## Bazı Komutların Kullanımı

**Şifre değiştirme**

```shell
passwd
```

---

**Sistemden çıkma**

```shell
exit or
logout or
Control + D

```

---

**Sistem değişkenleri**

```shell
set
```

---

**Dosya ve klasörleri listeleme**

```shell
ls
```

_Bazı seçenekler_

```shell
ls -a
ls -l
ls -al
ls -a /usr/folder
```
